Section 3 Present Tenses

When we talk about timeless truths, permanent situations or things that happen repeatedly, we usally use the simple present.

When we talk about temporary continuing actions and events, which are just going on now or around now, we usally use a present progressive tense.

Both present tenses can be used to talk about the future.
